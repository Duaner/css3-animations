Animations CSS3

Découvrez les bases et les secrets des animations CSS3.

Les animations, qu'est-ce que c'est ?

Les animations apparues avec CSS3, permettent de mettre en mouvement, ou de changer l'état initial d'un élément de la page. Les animations ont un grand rayon d'action, et permettent de changer beaucoup de propriétés CSS. Le grand absent, est très certainement la propriété display: block; qui n'est pas animable. On doit pour le moment passer par une sorte de "hack" pour la réaliser, en ajoutant un height: 0; opacity: 0; overflow: hidden.

Comment les créer ?

Création des « images-clés »

Cela se passe en deux étapes. En premier on créée une keyframe qui correspond aux images-clés de l'animation, c'est à dire les changements de propriétés/valeurs suivant le temps.

Toutes les propriétés CSS3 (ou presque) sont animables. Changement de couleurs, de fond, de taille de texte, de largeur d'un bloc, de hauteur. Comme je le disais tout, ou preque, puisque ce qui parait le plus utile et le plus important n'est pas encore animable, c'est la propriété display. Pour émuler ce comportement on est obligé de passer par un height: 0, opacity: 0; overflow: hidden.

Ci-dessous les « images-clés » pour l'animation de la bille :

@-webkit-keyframes animBille {
	100% { -webkit-transform: translateX(400px); }
}
@-moz-keyframes animBille {
	100% { -moz-transform: translateX(400px); }
}
@-o-keyframes animBille {
	100% { -o-transform: translateX(400px); }
}
@keyframes animBille {
	100% { transform: translateX(400px); }
}
